Oiseau is a modern DGTD framework for advanced simulations.

⚠️ APIs and features are subject to change.

🔧 Build Instructions

# Clone the repository
git clone https://github.com/tiagovla/oiseau

# Navigate into the project directory
cd oiseau

# Configure the project using CMake with Ninja
cmake -B build -G Ninja .

# Build the project
cmake --build build

For detailed installation and usage instructions, refer to the README on GitHub.
